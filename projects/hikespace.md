# Hikespace

#### Hiking-focused social media platform made with React and Express.js, entirely in TypeScript. Made for the January 2026 TechNative bootcamp.

#### Allows users to make posts containing images and location pins that outline the path hiked.

#### Utilises OpenStreetMap and Leaflet for the frontend to allow for location input, and Drizzle and PostGIS for database operations (and storage of location data) on the backend.

# cr0wbar's Rust Course

#### My final-year project at the University of Sussex. An interactive, educational web application that aims to teach the Rust programming language to beginner and mid-range developers in a concise, engaging and digestible manner.

#### Users can run code examples and observe the results to aid their understanding, take quizzes to test their knowledge and even write their own code in a sandboxed environment!
